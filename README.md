# AI Chatbot using Tkinter

This project is a simple AI chatbot desktop application built using Python and Tkinter. It connects to an API to generate responses based on user input.

## Features

* GUI-based chatbot using Tkinter
* Basic user input form (name, phone, email, state)
* Chat interface with scrollable message area
* API-based response generation

## Technologies Used

* Python
* Tkinter
* Requests

## Installation

1. Clone the repository:

git clone https://github.com/thahmeedha/chat_bot.git

2. Install dependencies:

pip install -r requirements.txt

## API Setup

Replace the placeholder API key in the code:

API_KEY = "YOUR_API_KEY_HERE"

You need to generate your own API key from the provider.

## Run the Project

python chatbot_app.py

## Notes

* The API key is not included for security reasons.
* Internet connection is required for chatbot responses.
